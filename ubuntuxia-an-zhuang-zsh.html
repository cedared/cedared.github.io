Ubuntu下安装zsh

Linux ?? Comments 四 10 四月 2014

Ubuntu下安装zsh，配置oh-my-zsh

一、首先安装zsh

1. 去zsh官网http://zsh.org下载最新版zsh安装包

2. 解压到任意目录

3. 编译安装

    $ ./configure
    $ make
    $ sudo make install

4. 将系统旧版本zsh替换

    $ sudo mv /usr/local/bin/zsh /usr/local/bin/zsh-old
    $ sudo mv /usr/local/bin/zsh-x.x.x /usr/local/bin/zsh

注：x.x.x为你下载安装的zsh版本号

5. 将zsh注册为系统shell

    $ sudo add-shell /usr/local/bin/zsh

二、 配置oh-my-zsh

1. 安装oh-my-zsh

    $ wget https://github.com/robbyrussell/oh-my-zsh/raw/master/tools/install.sh -O - | zsh

注： 这里会报错“密码： chsh: PAM authentication failed” 无需理会，只是将zsh切换为系统默认sh时出错，下一步我们手动切换就好。

2. 将系统默认shell设置为zsh

    $ chsh -s `which zsh`

3. 重启

    $ sudo reboot

4. 重启后配置oh-my-zsh

配置文件为~/.oh-my-zsh 这里不描述如何配置了，只把我的配置文件贴出来吧
